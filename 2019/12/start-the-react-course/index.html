Exports & Imports

I will want to split code across multiple files, but might still need to access functionality from a different module. This is where exporting and importing helps.

  • default 👉 export default ...;

  • named 👉 export const someData = ...;

  • import default exports with:
    import whatNameYouWantToUseHere from './file.js';

  • Or import names exports by exact name with braces
    import { someData } from './file.js';

A file can only contain one default and an unlimited amount of named exports. You can also mix the one default with any amount of named exports in one and the same file.

Array functions

Particularly important in this course are 👇

  • map() creates a new array with the results of a function on every element in the calling array
  • find() returns the first element in the provided array that satisfies a testing function
  • findIndex() returns the index of the first element in the array that satisfies the test
  • filter() creates a new array with all elements passing the test implemented by the function
  • reduce() executes a reducer function on each element, resulting in a single output value
  • concat() is used to merge two or more arrays, without changing the existing arrays
  • slice() returns a shallow copy of a portion of an array into a new array object
  • splice() changes contents of an array by removing, replacing and/or adding new elements
